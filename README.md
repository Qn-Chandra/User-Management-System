# 🚀 User Management System

A full-stack **User Management System** built using modern web technologies to efficiently manage user data with CRUD operations.

---

## 📌 Project Overview

This project is designed to perform complete **User Management** operations such as:

* ➕ Create Users
* 📄 Read/View Users
* ✏️ Update User Details
* ❌ Delete Users

It demonstrates full-stack development by integrating a **React frontend** with a **Flask backend API**.

---

## 🛠️ Tech Stack

### 💻 Frontend:

* React.js
* Axios
* HTML5, CSS3

### ⚙️ Backend:

* Flask (Python)
* REST API

### 🗄️ Database:

MongoDB

## ⚡ Installation & Setup

### 🔧 Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

### 💻 Frontend Setup

```bash
cd frontend
npm install
npm start
```

## 🌐 API Endpoints (Example)

| Method | Endpoint   | Description   |
| ------ | ---------- | ------------- |
| GET    | /users     | Get all users |
| POST   | /users     | Create user   |
| PUT    | /users/:id | Update user   |
| DELETE | /users/:id | Delete user   |
